## ❔ Sobre

Projeto criado na Trilha de React Native do Rocketseat Ignite. É uma aplicação para auxiliar no controle de gastos, onde o usuário pode adicionar ganhos e gastos com base em 6 categorias: salário, alimentação, carro, lazer, estudos e compras.


## 💻 Tecnologias usadas

- [React Native](https://reactnative.dev/)
- [Typescript](https://www.typescriptlang.org/)
- [Styled-component](https://styled-components.com/)
- [Expo](https://expo.io/) 


## 🛴 How to Rum

Antes de mais nada, é necessário ter o expo e o node instalados na sua máquina. Você pode adquirí-los aqui, caso não tenha:
- [Node](https://nodejs.org/en/)
- [Expo](https://expo.io/)

Procure baixar a versão LTS, que é a mais estável!

O aplicativo faz autenticação com o Google e a Apple. É necessário criar um projeto no [Google Platform Cloud](https://cloud.google.com/?hl=pt-br) e lá em API e Serviços, entre em Tela de permissão OAUth e coloque externo, além disso configure seus credenciais que são utilizadas no arquivo .env. Um adendo ao scope, que são usadas apenas de profile e email de autenticação.

Depois de clonar o projeto, substitua as informação no .env e baixe suas dependências rodando `yarn` ou `npm` e dê um `expo start`.
